# Repositório GiovanniJorge/HTML

## Sobre

Este repositório reúne exercícios e projetos desenvolvidos utilizando HTML, com objetivo de aprimorar habilidades na criação de páginas web estáticas, estruturação semântica e integração básica com CSS e JavaScript.

## Estrutura dos arquivos

- [`Trivia Game`](https://github.com/GiovanniJorge/HTML/blob/main/Trivia%20Game): Jogo de perguntas estilizado com elementos responsivos.

Além disso, o repositório inclui:
- [`.gitattributes`](https://github.com/GiovanniJorge/HTML/blob/main/.gitattributes): Configuração de atributos Git
- [LICENSE (MIT)](https://github.com/GiovanniJorge/HTML/blob/main/LICENSE): Licença MIT

## Como utilizar

1. Clone o repositório:
   ```bash
   git clone https://github.com/GiovanniJorge/HTML.git
   ```
2. Abra o arquivo `index.html` em seu navegador para visualizar o projeto.
3. Sinta-se à vontade para modificar os arquivos e testar novas ideias!
